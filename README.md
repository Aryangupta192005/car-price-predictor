# Car Price Predictor

A Machine Learning project to predict car prices using vehicle-related features such as mileage, fuel type, and manufacturing year.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## ML Models Used
- Linear Regression
- Random Forest Regressor
- Decision Tree

## Objective
To predict car prices using machine learning techniques and compare model performance.
